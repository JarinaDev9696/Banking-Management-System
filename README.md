# 🏦 Banking Management System – SQL Mini Project

## 📌 Project Overview
Banking Management System developed using SQL Server (SSMS).
4 tables and 17 SQL queries for real-world banking scenarios.

## 🗄️ Tables
- Customers
- Accounts
- Transactions
- Loans

## 🔧 SQL Concepts Used
- INNER JOIN & LEFT JOIN
- Aggregate Functions: SUM(), AVG(), COUNT(), MIN(), MAX()
- GROUP BY & HAVING
- Subqueries
- Window Functions: RANK(), DENSE_RANK(), LAG(), LEAD(), SUM OVER()
- TOP N & ORDER BY

## 📊 Key Features
- Display customer account details with balances
- Find top customers based on highest balance
- Analyze deposits and withdrawals
- Track customer transactions
- Customer ranking and running total analysis

## 🛠️ Tools Used
- SQL Server Management Studio (SSMS)

## 🎯 Learning Outcomes
Hands-on experience in database design, writing optimized
SQL queries, and applying advanced SQL concepts.
